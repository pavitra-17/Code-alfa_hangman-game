# Code-alfa_hangman-game# Hangman Game Project

This is a simple Python Hangman Game where the player guesses letters to find the hidden word before running out of chances.

## Features
- Random word selection
- Limited guessing attempts
- Tracks correct and wrong guesses
- Beginner-friendly game project
- Command-line based interface

## Technologies Used
- Python
- Random Module

## How to Run
1. Install Python
2. Open terminal or command prompt
3. Run the program:
```bash
python hangman_game.py
